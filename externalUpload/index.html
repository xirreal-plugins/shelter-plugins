External Upload

File too big? Discord lowering the upload limit once again? Just upload it somewhere else!

Requires an S3 compatible bucket to work. I use this with Cloudflare R2 since the usage limits are realistically high enough to not get billed. I decline responsibility for any broken wallets in case you do. Rememeber to set up CORS to allow uploads from discord.com!
This plugin is *somewhat* mindful of privacy, hashing filenames and storing them as private S3 Object metadata to mitigate scraping attempts, but ultimately you're uploading your files to a public bucket. Don't upload anything you wouldn't want to be public.
